# Rankings Map

Компилятор международных рейтингов на мировой карте с цветовым градиентом и
движком пользовательских формул. Берёшь готовые индексы по странам, пишешь
формулу — получаешь новый составной индекс и видишь его choropleth-картой.

**Static-first**: ноль серверов в рантайме. Данные собираются на этапе билда в
статические артефакты, формулы считаются в браузере (Web Worker).

## Возможности

- Выбор готового индекса или **своя формула** (`life_exp * 0.1 + gdp_pc`), движок
  на `expr-eval` (безопасный AST, без `eval`) в Web Worker.
- Нормализация входов: нет / z-score / min-max; учёт полярности индекса
  (`higherBetter` / `lowerBetter`).
- Слайдер года (2000–2022), легенда, тултипы, цветовая шкала viridis (D3).

## Данные

Сейчас подключено 3 индекса World Bank (CC BY 4.0): ВВП на душу (`gdp_pc`),
ожидаемая продолжительность жизни (`life_exp`), CO₂ на душу (`co2_pc`),
217 стран, 2000–2022.

## Разработка

```bash
npm install
npm run dev        # дев-сервер Vite
npm run build      # tsc --noEmit + vite build → dist/
npm run etl        # пересобрать public/data/dataset.json из World Bank API
```

Рендер карты в PNG без браузера: `node scripts/render.mjs`.

## Деплой

Автодеплой на **GitHub Pages** через GitHub Actions на каждый push в `main`
(`.github/workflows/deploy.yml`). Сайт собирается из закоммиченного
`public/data/` — World Bank во время деплоя не дёргается.

## Лицензия

Код — на усмотрение автора. Данные — World Bank, CC BY 4.0 (атрибуция в карточке
источника на карте).
